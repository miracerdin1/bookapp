Kısa açıklama
-------------
BookApp; Vue 3, Vite ve TypeScript kullanılarak oluşturulmuş basit bir kitap takip uygulamasıdır. Hızlı geliştirme için Vue 3,  Vite, TypeScript,  Pinia, Tailwind, SCSS ile birlikte gelir.
Proje Link: [Reperi book app](https://book-app-reperi-ai.netlify.app/)

Kurulum ve çalıştırma
---------------------
1. Depoyu klonlayın:
   ```bash
   git clone https://github.com/miracerdin1
   cd BookApp
-------------
1-Bağımlılıkları yükleyin:
    ```bash
    npm install
    ```
2- Geliştirme sunucusunu çalıştırın:
    ```bash
    npm run dev
    ```

Özellikler
-------------
Vue 3 + Composition API
TypeScript desteği
Vite hızlı geliştirme ortamı
Pinia ile durum yönetimi
Tailwind CSS + SCSS ile stil yapısı

Potansiyel/eklenmiş uygulama özellikleri:
--------------
Kitap ekleme, düzenleme ve silme
Arama ve filtreleme
Yerel depolama (localStorage) veya basit bir backend ile eşitleme
Kolay test desteği (Vitest/Jest entegrasyonu eklenebilir)
Responsif tasarım için hazır Tailwind sınıfları
